# Vibe Coding 行业趋势研究报告

> 调研日期：2026-03-20

---

## 一、什么是 Vibe Coding

**定义：** Vibe Coding（氛围编程）是一种用自然语言描述需求、由 AI 生成代码的软件开发方式。开发者的角色从"写语法"转变为"描述意图 → 审查输出 → 迭代优化"。

**起源：**
- 由 Andrej Karpathy（前 Tesla AI 总监、OpenAI 联合创始成员）于 **2025 年 2 月**在 X (Twitter) 上首次提出
- 原话核心思想："fully give in to the vibes, embrace exponentials, and forget that the code even exists"
- 2025 年被 Collins Dictionary 评为**年度词汇**，搜索量暴涨 **6,700%**

**当前状态（2026年3月）：**
- 已从概念演变为**真实职业**——Lovable 等平台已在招聘专职 "vibe coder"
- Business Insider 报道：越来越多人全职从事 vibe coding，无需掌握传统编程语言
- Forbes 报道：vibe coding 相关岗位薪资可达 **$220,000/年**

## 三、Vibe Coding 交易市场（Marketplace）

### 3.1 传统自由职业平台的渗透

| 平台 | 现状 |
|------|------|
| **Upwork** | 已开设 "Vibe Coding Developers" 专属分类，可直接搜索和雇佣 vibe coder |
| **Fiverr** | 已有 24+ 个 AI Vibe Coding 服务在售；出现专职 vibe coding 卖家 |

### 3.2 新兴垂直市场平台

| 平台 | 定位 | 模式 |
|------|------|------|
| **Greta Vibe Marketplace** | 创作者出售 app 模板、UI 组件、AI prompt | 250+ app 模板，100+ UI 组件，350+ prompt；创作者可持续赚取收入 |
| **AppVibed** | Vibe Coding 成品应用市场 | 买断制，可编辑代码，一次付费 |
| **Vibe Coded** | 移动应用交易市场 | 买卖预构建的 vibe coded 项目 |
| **VibeMarketplace.io** | Vibe-coded 创业项目买卖 | 经验证收入的创业项目交易，类似 Acquire.com |
| **GoodVibeCode.com** | Vibe Coding 求职平台 | 专门聚合 AI 辅助开发岗位 |
| **RemoteVibecodingJobs.com** | 远程 Vibe Coding 工作 | 远程 AI 辅助开发者岗位 |

### 3.3 关键洞察

- Moltbook 案例：创始人 Matt Schlicht 用 AI 编码工具**零代码**构建社交网络，4天内获得 **150万 AI Agent 用户**
- Lovable 付费用户中大部分是**创始人和创业者**，包括 Klarna、Uber、Deutsche Telekom 等企业客户（32万+付费用户）
- Solo founder 用 vibe coding 可在 **72小时内** 发布 MVP

## 四、"一人公司"趋势

**核心论点：** Anthropic CEO 预测，到 2026 年，1-2 个人可以运营**十亿美元级公司**。

**现状：**
- "Solopreneur Tech Stack" 已成为 2026 年热门话题
- 典型技术栈：Cursor/Claude Code（编码）+ Lovable/Bolt（快速原型）+ Vercel（部署）
- 从想法到 MVP 的周期从数月压缩到**数天甚至数小时**
- Nucamp 等教育平台已开设"如何用 vibe coding 构建并变现项目"课程

**代表性工具组合：**
- IDE 层：Cursor、Windsurf、Claude Code
- Builder 层：Lovable、Bolt.new、v0、Replit
- 部署层：Vercel、Netlify
- 后端/数据：Supabase、Firebase

## 五、中国市场竞争格局

### 5.1 术语与社区

- 中文译名：**氛围编程**（fēnwéi biānchéng）
- 小红书、即刻等平台上已形成活跃的氛围编程讨论社区
- 关键话题标签：#氛围编程 #AI编程 #vibe coding #一人公司 #独立开发者

### 5.2 中国本土平台与工具

| 类别 | 代表 |
|------|------|
| **AI 编程工具** | 通义灵码（阿里）、CodeGeeX（智谱）、Marscode（字节跳动）、Lingma |
| **社区/教育** | AI聘（aipin.io）举办 Vibe Coding 活动；北京信息科技大学等高校开设相关课程 |
| **跨境出海** | 大数跨境等平台关注 vibe coding 在跨境电商中的应用 |

### 5.3 中国市场特点

- **教育驱动**：高校和培训机构快速跟进，台大已开设免费 Vibe Coding 课程
- **独立开发者生态**：即刻社区的独立开发者群体与 vibe coding 高度重合
- **变现路径**：小红书上出现大量"AI编程接单""零代码赚钱"内容
- **工具本土化**：字节、阿里、智谱等大厂均推出本土 AI 编程助手，但尚未出现对标 Lovable/Bolt 的本土 AI App Builder

### 5.4 中国市场空白

- **缺少垂直交易平台**：没有对标 Greta Marketplace、AppVibed、VibeMarketplace.io 的中文平台
- **缺少 vibe coder 接单平台**：Upwork/Fiverr 已有 vibe coding 分类，国内猪八戒、程序员客栈等尚未跟进
- **缺少 vibe coded 产品交易市场**：没有买卖 AI 生成应用/模板的中文市场

## 六、市场机会总结

### 6.1 数据支撑

| 指标 | 数据 |
|------|------|
| 全球 AI 编程助手市场规模 | $85亿（2026年） |
| Cursor 增速 | 史上最快 SaaS 公司，26个月 $0→$20亿 ARR |
| Lovable 增速 | 单月 ARR 增长 33%，当前 $4亿 |
| Vibe coding 搜索增长 | 6,700% |
| 自由职业市场渗透 | Upwork/Fiverr 已开设专属分类 |

### 6.2 核心机会窗口

1. **Vibe Coder 服务市场（中国版）**：连接"会用 AI 工具的开发者"与"有需求的客户"，类似 Upwork 的 vibe coding 分类，但本土化
2. **Vibe Coded 产品交易市场（中国版）**：买卖 AI 生成的应用、模板、组件，对标 AppVibed / Greta Marketplace
3. **一人公司孵化平台**：为 solopreneur 提供从 idea → MVP → 变现的全链路支持
4. **垂直行业 Vibe Coding 服务**：针对电商、小程序、企业内部工具等中国特色场景

### 6.3 竞争壁垒思考

- 工具层已被巨头占据（Cursor $293亿、Lovable $66亿、Replit $90亿）
- **平台/市场层仍处早期**——AppVibed、Greta、VibeMarketplace 均为初创项目
- 中国市场几乎**完全空白**，存在先发优势窗口
- 关键差异化：质量审核、交付保障、本土支付/合规
