2. **交付质量=口碑** —— 每一次优质交付都产生评价、作品案例和口碑传播素材
3. **UGC自传播** —— 用户主动分享"花800块做了个APP"的故事，成为最好的获客内容
4. **双边网络效应** —— 需求越多→造物者越活跃→交付越快→满意度越高→口碑越好→需求越多

| 需求发布量 | 当月新发布的需求数 | 80+ |

| 匹配→成交率 | 匹配成功后达成交易的比例 | >40% |
| 交付完成率 | 开始项目后成功交付验收的比例 | >85% |
| NPS（净推荐值） | 用户推荐意愿评分 | >40 |
